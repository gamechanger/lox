post /lock

Attempt to acquire a shared lock.

  • Request
  • Response

Query Parameters

  • key required (string)

    An identifier for the shared lock you want to acquire

    Example: lox.shared.lock.123

  • maximumHeldKeys required (integer)

    The maximum allowable number of currently held keys. If this many keys are already held, acquisition will fail

    Example: 5

  • ttlSeconds required (number)

    The number of seconds for which to hold the lock

    Example: 60

HTTP status code 201

Successfully acquired a shared lock

Body

Type: application/json

Example: 
{"lockId": "3885b1e0-022c-11e4-b7c7-6b932dd1bfa1"}
